## Service Positioning & Responsibility Boundaries

### Platform Positioning

LaoZhang API as a **technical API integration service provider**:

- **Service Nature**: Provides technical integration capabilities for AI model APIs
- **Technical Channel Positioning**: Only provides technical transmission channels for API request forwarding
- **No Content Decision-Making**: Does not decide what content to generate or interfere with generation logic
- **No User Content Storage**: Does not cache, analyze, or reuse user data

### Core Positioning (3 Key Points)

<Info>
**Technical Service Positioning Statement**

1. Data is uploaded at the customer's own discretion
2. We do not process, store, or analyze data content
3. We only provide technical API transmission capabilities
</Info>

### User Responsibility

<Warning>
**User Notice**

When using this service, users must bear the following responsibilities:

1. **Content Legality**: Ensure input content and usage purposes comply with laws and regulations
2. **Output Responsibility**: Be responsible for AI-generated output content
3. **Usage Scenarios**: Ensure usage scenarios are legal and compliant
4. **Local Regulations**: Comply with laws and regulations of your region
5. **Data Authorization**: Confirm authorization to transmit relevant data to model service providers

**This platform is not responsible for user behavior and generated content.**
</Warning>

### Prohibited Uses

This service **must not be used for**:

- Automated content generation services for the public
- Content production that violates laws and regulations
- Actions that infringe on the rights of others
- Any illegal or harmful purposes

The following types of content are strictly prohibited:
