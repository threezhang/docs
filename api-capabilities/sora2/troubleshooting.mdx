## 视频生成相关

<AccordionGroup>
  <Accordion title="如何控制视频的横屏还是竖屏？" icon="mobile-screen">
    **通过模型名控制**，而不是参数。

    - **竖屏：** `sora_video2`, `sora_video2-hd`
    - **横屏：** `sora_video2-landscape`, `sora_video2-landscape-hd`

    示例：
    ```python
    # 竖屏视频
    model = "sora_video2"

    # 横屏视频
    model = "sora_video2-landscape"
    ```

    详见 [模型与定价](/api-capabilities/sora2/models-pricing) 页面。
  </Accordion>

  <Accordion title="如何生成 15 秒视频？" icon="clock">
    在模型名后加 `-15s` 后缀：

    ```python
    # 10秒竖屏（标准）
    model = "sora_video2"

    # 15秒竖屏
    model = "sora_video2-15s"

    # 15秒横屏高清
    model = "sora_video2-landscape-hd-15s"
    ```

    **价格：** $0.2/次（15秒），$0.15/次（10秒）
  </Accordion>

  <Accordion title="生成的视频有水印吗？" icon="droplet">
    **没有水印！**

    本站提供的 Sora 2 API 生成的视频**无水印**，而官网生成的视频仍然带有 Sora 水印。

    <Note>
    原先的无水印方案因 OpenAI 升级已失效，现已更新为新的无水印方案。
    </Note>
  </Accordion>

  <Accordion title="为什么生成人物视频失败？" icon="user-xmark">
    **限制原因：**

    1. **真实人脸参考图会被拒绝** - 不支持上传真人照片
    2. **仅支持授权真人** - 通过 `@ID` 方式使用

    **可用的授权真人：**
    - `@sama` - OpenAI CEO Sam Altman

    **正确示例：**
    ```python
    prompt = "@sama 在长城上开心地说话"  # ✓ 可以
    ```

    **错误示例：**
    ```python
    # ✗ 不可以 - 上传真人照片会被拒绝
    # 上传了一张真人照片 + 提示词
    ```
  </Accordion>

  <Accordion title="生成时间太长怎么办？" icon="hourglass">
    **正常生成时间：**
    - 排队：视高峰期而定
    - 生成：2-3 分钟
    - 总计：2.5-4 分钟

    **优化建议：**

    1. **设置合理超时：** 建议 5 分钟（300秒）
    ```python
    import httpx

    client = openai.OpenAI(
        api_key="YOUR_API_KEY",
        base_url="https://api.laozhang.ai/v1",
        http_client=httpx.Client(timeout=300.0)
    )
    ```

    2. **使用流式输出：** 实时查看进度
    ```python
    response = client.chat.completions.create(
        model="sora_video2",
        messages=[...],
        stream=True  # 启用流式输出
    )
    ```

    3. **避开高峰期：** 选择使用人数较少的时段
  </Accordion>

  <Accordion title="横屏模型为什么经常失败？" icon="tv">
    **稳定性说明：**

    横屏模型（`sora_video2-landscape` 系列）是新增功能，目前稳定性一般，可能需要重试。

    **稳定性对比：**
    - 竖屏标准版 `sora_video2`：⭐⭐⭐⭐⭐ 高
    - 竖屏高清版 `sora_video2-hd`：⭐⭐⭐⭐ 较高
    - 横屏模型：⭐⭐⭐ 一般

    **解决方案：**
    1. 重试 2-3 次
    2. 优化提示词，使描述更清晰
    3. 或改用竖屏模型
  </Accordion>

  <Accordion title="报错：We're under heavy load, please try again later" icon="server">
    **原因：** OpenAI 官方负载过高

    **解决方案：**
    1. 等待几分钟后重试
    2. 添加重试逻辑
    ```python
    import time

    max_retries = 3
    for i in range(max_retries):
        try:
            response = client.chat.completions.create(...)
            break
        except Exception as e:
            if "heavy load" in str(e) and i < max_retries - 1:
                print(f"服务繁忙，{30}秒后重试...")
                time.sleep(30)
            else:
                raise
    ```
  </Accordion>
</AccordionGroup>

  <Accordion title="如何查看生成进度？" icon="spinner">
    **使用流式输出：**

    ```python
    stream = client.chat.completions.create(
        model="sora_video2",
        messages=[...],
        stream=True  # 启用流式输出
    )

    for chunk in stream:
        if chunk.choices[0].delta.content:
            print(chunk.choices[0].delta.content, end='', flush=True)
    ```

    **进度信息示例：**
    ```
    > ⌛️ 任务正在队列中，请耐心等待...

    > 🏃 进度：36.0%

    > 🏃 进度：68.5%

    > ✅ 视频生成成功，[点击这里](https://xxx.mp4) 查看视频~~~
    ```
  </Accordion>

  <Accordion title="如何处理超时错误？" icon="clock-rotate-left">
    **增加超时时间：**

    ```python
    import httpx
    import openai

    # 设置 5 分钟超时
    client = openai.OpenAI(
        api_key="YOUR_API_KEY",
        base_url="https://api.laozhang.ai/v1",
        http_client=httpx.Client(timeout=300.0)
    )
    ```

    **建议超时时间：**
    - 最小：5 分钟（300秒）
    - 推荐：10 分钟（600秒）
  </Accordion>

  <Accordion title="支持哪些图片格式？" icon="image">
    **图生视频支持：**

    1. **URL 图片**
    ```python
    {
        "type": "image_url",
        "image_url": {
            "url": "https://example.com/image.png"
        }
    }
    ```

    2. **Base64 编码**
    ```python
    {
        "type": "image_url",
        "image_url": {
            "url": "data:image/png;base64,iVBORw0KG..."
        }
    }
    ```

    **支持的格式：**
    - PNG
    - JPEG/JPG
    - WebP
    - GIF（会使用第一帧）

    **限制：**
    - 最多 1 张图片
    - 建议分辨率不超过 2048×2048
  </Accordion>
</AccordionGroup>

## 视频下载相关

<AccordionGroup>
  <Accordion title="视频链接有效期多久？" icon="link-slash">
    **有效期：仅 1 天**

    <Warning>
    视频存储在国内 CDN，**存储时效仅 1 天**。请在生成后**立即下载**保存到本地！
    </Warning>

    **最佳实践：**
    ```python
    import requests

    def download_video(url, save_path):
        """下载视频到本地"""
        response = requests.get(url, stream=True)
        with open(save_path, 'wb') as f:
            for chunk in response.iter_content(chunk_size=8192):
                f.write(chunk)
        print(f"已保存：{save_path}")

    # 生成后立即下载
    video_url = extract_video_url(result)
    download_video(video_url, "my_video.mp4")
    ```
  </Accordion>

  <Accordion title="如何提取视频链接？" icon="link">
    **从响应中提取：**

    ```python
    import re

    def extract_video_url(content):
        """从响应内容中提取视频链接"""
        match = re.search(r'https://[^\s\)]+\.mp4', content)
        return match.group(0) if match else None

    # 使用示例
    response = client.chat.completions.create(...)
    content = response.choices[0].message.content
    video_url = extract_video_url(content)
    ```

    **链接格式示例：**
    ```
    https://sora.gptkey.asia/assets/sora/xxx.mp4
    ```
  </Accordion>

  <Accordion title="下载速度慢怎么办？" icon="gauge-simple">
    **优化建议：**

    1. **使用国内网络** - 视频存储在国内 CDN，国内访问快
    2. **流式下载** - 避免一次性加载到内存
    ```python
    import requests

    response = requests.get(video_url, stream=True)
    with open('video.mp4', 'wb') as f:
        for chunk in response.iter_content(chunk_size=8192):
            f.write(chunk)
    ```
    3. **断点续传** - 使用支持断点续传的下载工具
  </Accordion>
</AccordionGroup>
