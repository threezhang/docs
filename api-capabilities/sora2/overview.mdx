## 前置要求

<Steps>
  <Step title="获取 API Key">
    登录 [laozhang.ai 控制台](https://api.laozhang.ai) 获取 API 密钥
  </Step>
  <Step title="配置计费模式">
    编辑令牌设置，选择以下任一计费模式（两者价格相同）：

    - **按量优先**（推荐）：优先使用余额计费，余额不足时自动切换。适合大多数用户
    - **按次计费**：每次调用直接扣费。适合预算控制严格的场景

    <Note>
    两种模式**价格完全相同**，都是 \$0.15/次（10秒或15秒），仅扣费方式不同。
    </Note>

    <img src="/images/sora2-token-setting.png" alt="令牌设置" />

    <Warning>
    如果未设置计费模式，API调用会失败。必须先完成此配置！
    </Warning>
  </Step>
</Steps>

## 工作流程

<Steps>
  <Step title="配置 API Key">
    在 laozhang.ai 控制台获取 API 密钥，设置为**按量优先**或**按次计费**模式
  </Step>
  <Step title="发送请求">
    调用 `/v1/chat/completions` 端点，指定模型和提示词
  </Step>
  <Step title="实时进度">
    通过流式输出查看任务队列和生成进度
  </Step>
  <Step title="获取视频">
    生成完成后获取视频链接（有效期 1 天）
  </Step>
  <Step title="下载保存">
    立即下载视频到本地保存
  </Step>
</Steps>

## 调用方式选择

<Tip>
**推荐使用异步 API**，特别是在生产环境或需要批量生成视频时，稳定性更有保障。
</Tip>

| 特性 | 同步 API | 异步 API（推荐）|
|------|----------|-----------------|
| **端点** | `/v1/chat/completions` | `/v1/videos` |
| **稳定性** | 依赖长连接 | ⭐⭐⭐⭐⭐ 更稳定 |
| **适用场景** | 快速测试、实时反馈 | 生产环境、批量生成 |
