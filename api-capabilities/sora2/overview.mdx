## 工作流程

<Steps>
  <Step title="配置 API Key">
    在 laozhang.ai 控制台获取 API 密钥，设置为**按量优先**或**按次计费**模式
  </Step>
  <Step title="发送请求">
    调用 `/v1/chat/completions` 端点，指定模型和提示词
  </Step>
  <Step title="实时进度">
    通过流式输出查看任务队列和生成进度
  </Step>
  <Step title="获取视频">
    生成完成后获取视频链接（有效期 1 天）
  </Step>
  <Step title="下载保存">
    立即下载视频到本地保存
  </Step>
</Steps>

## 支持的模型

### 标准版（10秒视频）

| 模型 | 分辨率 | 价格 | 稳定性 | 说明 |
|------|--------|------|--------|------|
| `sora_video2` | 704×1280 | $0.15/次 | 高 | 竖屏，默认推荐 |
| `sora_video2-hd` | 1024×1792 | $0.15/次 | 较高 | 竖屏高清版 |
| `sora_video2-landscape` | 1280×704 | $0.15/次 | 一般 | 横屏标准版 |
| `sora_video2-landscape-hd` | 1792×1024 | $0.15/次 | 一般 | 横屏高清版 |

### 15秒视频版本

在任意模型名后加 `-15s` 后缀：
- `sora_video2-15s`
- `sora_video2-hd-15s`
- `sora_video2-landscape-15s`
- `sora_video2-landscape-hd-15s`

**价格：** $0.2/次

## 视频规格

<Info>
- **时长：** 10秒（标准）/ 15秒（加 -15s 后缀）
- **清晰度：** 720P / 高清版更高
- **默认方向：** 竖屏（可选横屏模型）
- **水印：** 无水印
- **音频：** 支持音视频同步
</Info>
