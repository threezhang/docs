## 工作流程

<Steps>
  <Step title="配置 API Key">
    在 laozhang.ai 控制台获取 API 密钥，设置为**按量优先**或**按次计费**模式
  </Step>
  <Step title="发送请求">
    调用 `/v1/chat/completions` 端点，指定模型和提示词
  </Step>
  <Step title="实时进度">
    通过流式输出查看任务队列和生成进度
  </Step>
  <Step title="获取视频">
    生成完成后获取视频链接（有效期 1 天）
  </Step>
  <Step title="下载保存">
    立即下载视频到本地保存
  </Step>
</Steps>

<Note>
**模型暂时精简通知**

由于官方 OpenAI 算力不够，限额严重，-hd（高清）和 -15s（15秒）版本暂时不开放，等后续算力充足后会重新上线。

当前仅提供两个稳定性极高的模型供选择。
</Note>
