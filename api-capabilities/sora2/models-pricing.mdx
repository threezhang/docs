## 模型概览

Sora 2 提供多种模型以满足不同的使用场景，主要通过**模型名**来区分横竖屏和清晰度。

<Note>
**特别说明**

与传统 API 不同，Sora 2 通过**模型名**（而非参数）来控制：
- 视频方向（横屏/竖屏）
- 清晰度（标准/高清）
- 时长（10秒/15秒）
</Note>
