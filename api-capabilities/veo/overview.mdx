## 概述

VEO API 提供强大的 AI 视频生成能力，支持 VEO3 系列的 4 种不同模型。服务采用异步处理模式，确保高效处理视频生成请求。

## 工作流程

<Steps>
  <Step title="提交任务">
    调用提交接口，发送文本描述和参数
  </Step>
  <Step title="获取任务 ID">
    系统返回唯一的任务 ID
  </Step>
  <Step title="轮询状态">
    使用任务 ID 查询处理进度
  </Step>
  <Step title="获取结果">
    任务完成后获取视频 URL
  </Step>
</Steps>
