## 简介

Veo-3.1 是 Google 推出的最新一代视频生成模型系列,支持通过文本描述或图片输入生成高质量视频内容。该系列模型完全兼容 OpenAI Chat Completions API 格式,方便开发者快速集成。

## 生成方式

### 文生视频（Text-to-Video）

通过自然语言描述生成视频内容:

```json
{
  "messages": [{
    "role": "user",
    "content": [
      {
        "type": "text",
        "text": "生成两只猫和一只狗打架的视频"
      }
    ]
  }],
  "model": "veo-3.1",
  "stream": true,
  "n": 2
}
```

### 图生视频（Image-to-Video）

基于参考图片生成视频内容:

```json
{
  "messages": [{
    "role": "user",
    "content": [
      {
        "type": "text",
        "text": "根据两张图片生成一个完整的过渡视频"
      },
      {
        "type": "image_url",
        "image_url": {
          "url": "开始帧图片URL或base64"
        }
      },
      {
        "type": "image_url",
        "image_url": {
          "url": "结束帧图片URL或base64（可选）"
        }
      }
    ]
  }],
  "model": "veo-3.1-fl",
  "stream": true
}
```

## 核心参数说明

<ParamField path="model" type="string" required>
  选择使用的 Veo-3.1 模型变体
</ParamField>

<ParamField path="messages" type="array" required>
  包含用户输入的消息数组,支持文本和图片类型
</ParamField>

<ParamField path="stream" type="boolean" default="false">
  是否启用流式响应,建议设置为 `true` 以实时获取进度
</ParamField>

<ParamField path="n" type="integer" default="1">
  生成视频的数量,范围 1-4,可同时生成多个不同结果
</ParamField>

## 计费说明

<Warning>
**重要提示:** Veo-3.1 模型需要使用**按次付费令牌**,不支持按量付费令牌。请在 [老张API控制台](https://api.laozhang.ai/token) 创建"按次付费"类型的令牌。
</Warning>

<Info>
**按次计费：** 每次成功的 API 调用按所选模型的单价计费,失败不收费
</Info>

### 计费规则

- **单次调用：** 按模型价格计费（$0.15 或 $0.25）
- **多结果生成：** 设置 `n=2` 会生成 2 个视频,按 2 次计费
- **失败重试：** 生成失败不收费,可免费重试
- **流式响应：** 不额外收费
