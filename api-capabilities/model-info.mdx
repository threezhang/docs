## 🛠️ 使用建议

### 模型选择指南

**编程开发** 💻
- 首选：Claude 4.5 Sonnet、GPT-5 Codex、o4-mini (2025-04)
- 备选：Claude 4 Sonnet、DeepSeek V3.1、Gemini 2.5 Pro、Grok 4
- 经济型：GPT-5 Nano、DeepSeek V3.2 Exp

**文本创作** ✍️
- 首选：GPT-5、GPT-5 Pro、Claude 4.5 Sonnet
- 备选：Claude 4.1 Opus、GPT-4.1、Gemini 2.5 Pro
- 中文优化：Kimi K2 正式版、Qwen Max

**复杂推理** 🧠
- 首选：o3 (2025-04)、o3 Pro (2025-06)、GPT-5
- 备选：o3 Deep Research、Claude 4.5 Sonnet Thinking
- 轻量级：o3 Mini、o4-mini、DeepSeek R1 (05-28)

**快速响应** ⚡
- 首选：Gemini 2.5 Flash (05-20)、GPT-5 Mini、Grok 4 Fast
- 备选：Gemini 2.5 Flash Lite、GPT-4o Mini、Claude Haiku 4.5
- 经济型：GPT-3.5 Turbo、Grok 3 Mini Fast

**图像生成** 🎨
- 稳定优先：GPT-Image-1（官方定价高，本站价格八折）
- 质量优先：DALL·E 3
- 性价比（对话补全端口）：Sora Image、GPT-4o Image

**视频生成** 🎬
- 专业级：Sora 2 Pro、Veo 3 Pro
- 标准版：Sora Video 2、Veo 3
- 快速版：Veo 3 Fast
- 短视频：Sora Video 2 (15s)

**长文本处理** 📄
- 首选：Gemini 2.5 Pro（2M上下文）
- 备选：Gemini 1.5 Pro（2M上下文）
- 高质量：Claude 4.5 Sonnet（200K上下文）

**联网搜索** 🔍
- 深度搜索：Grok 3 DeepSearch（按次计费）
- o3 Deep Research（深度研究专用）

### 成本优化建议

1. **分级使用**：简单任务用便宜模型，复杂任务用高级模型
2. **测试优化**：先用小模型测试，确定需求后再用大模型
3. **批量处理**：大量相似任务可以选择 Nano 或 Mini 版本
4. **缓存复用**：对重复查询结果进行缓存
