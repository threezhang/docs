## 🛠️ 使用建议

### 模型选择指南

**编程开发**
- 首选：Claude 4 Sonnet、DeepSeek V3.1、o4-mini、Gemini 2.5 Pro
- 备选：DeepSeek Coder、Kimi K2 正式版、Qwen 3

**文本创作**
- 首选：GPT-4.5（文科强，但很贵）、Claude 4 Opus（也不便宜的模型）
- 备选：GPT-4o、chatgpt-4o-lastest、Kimi K2 正式版、Qwen Max

**快速响应**
- 首选：GPT-3.5 Turbo、Gemini 2.5 Flash
- 备选：Claude 3 Haiku、Grok 3 Mini

**图像生成**
- 稳定优先：GPT-Image-1（官方定价高，本站价格八折）
- 质量优先：DALL·E 3
- 性价比（对话补全端口）：Sora Image、GPT-4o Image

**长文本处理**
- 首选：Gemini 2.5 Pro（2M上下文）
- 备选：Claude 4 系列（200K上下文）

### 成本优化建议

1. **分级使用**：简单任务用便宜模型，复杂任务用高级模型
2. **测试优化**：先用小模型测试，确定需求后再用大模型
3. **批量处理**：大量相似任务可以选择 Nano 或 Mini 版本
4. **缓存复用**：对重复查询结果进行缓存
