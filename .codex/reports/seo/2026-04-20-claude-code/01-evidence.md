### Official-claude

- `https://code.claude.com/docs/en/overview`
  - Claude Code is described as an agentic coding tool that reads codebases, edits files, runs commands, and integrates with developer tools.
  - Current docs say most surfaces require a Claude subscription or Anthropic Console account; Terminal CLI and VS Code also support third-party providers.
  - Current install path lists native install, Homebrew, and WinGet, with native installers auto-updating.
- `https://code.claude.com/docs/en/quickstart`
  - Quickstart requires account login; supported account types include Pro/Max/Team/Enterprise, Claude Console, and cloud providers.
  - First real workflow: ask about codebase, make a change, approve edits, use git, run tests.
- `https://code.claude.com/docs/en/authentication`
  - Team authentication routes include Claude for Teams/Enterprise, Claude Console, Bedrock, Vertex, and Foundry.
  - Credential precedence matters: cloud provider vars, `ANTHROPIC_AUTH_TOKEN`, `ANTHROPIC_API_KEY`, apiKeyHelper, OAuth token, subscription OAuth.
  - Docs warn that an API key can take precedence over an active subscription and cause failures if tied to a disabled/expired organization.
- `https://claude.com/product/claude-code`
  - Current product page lists Pro, Max 5x, Max 20x, Claude API, Team, and Enterprise routes.
  - Current visible pricing: Pro $17/month annual or $20 monthly; Max 5x $100/month; Max 20x $200/month; Claude API pay-as-you-go.
  - FAQ says Claude Code can be accessed with Pro/Max, Team/Enterprise, or Claude Console credentials.
- `https://claude.com/pricing`
  - Pricing page says Pro includes Claude Code; Max adds 5x or 20x more usage than Pro.
  - Team standard and premium seats are listed, with Claude Code included in Team feature tables.
- `https://code.claude.com/docs/en/costs`
  - `/cost` is intended for API users, not Pro/Max subscription billing.
  - Console authentication creates a Claude Code workspace for centralized cost tracking.
