## 为什么国内开发者需要API中转服务？

**ChatGPT国内能用吗？Claude国内怎么访问？** 这是很多开发者的疑问。

### 🚀 三分钟快速接入

### 第一步：获取API密钥
注册账号后，在控制台获取您的API密钥

### 第二步：修改请求地址
```python
# 原代码（OpenAI官方）
client = OpenAI(
    api_key="sk-xxx",
    base_url="https://api.openai.com/v1"  # 官方地址
)

# 修改后（老张API）
client = OpenAI(
    api_key="您的密钥",
    base_url="https://api.laozhang.ai/v1"  # 改为老张API地址
)
```

### 第三步：调用任意模型
```python
# 一个接口，调用所有模型
models = [
    "gpt-4.1",               # OpenAI最新模型
    "claude-sonnet-4-20250514",  # Claude 4最新版
    "gemini-2.5-pro",        # Google Gemini
    "deepseek-v3"            # 国产最强模型
]

for model in models:
    response = client.chat.completions.create(
        model=model,
        messages=[{"role": "user", "content": "你好"}]
    )
    print(f"{model}: {response.choices[0].message.content}")
```
