## 企业级AI技术API接入平台

老张API提供**统一的AI模型API接入能力**，面向企业系统集成、软件开发及业务自动化场景：

- **统一API接口**：一套标准接口对接多个AI模型服务商
- **面向企业与开发者**：用于系统集成、应用开发、内部工具

## 快速接入示例

### 第一步：获取API密钥
注册账号后，在控制台获取您的API密钥

### 第二步：修改请求地址
```python
# 原代码（OpenAI官方）
client = OpenAI(
    api_key="sk-xxx",
    base_url="https://api.openai.com/v1"  # 官方地址
)

# 修改后（老张API）
client = OpenAI(
    api_key="您的密钥",
    base_url="https://api.laozhang.ai/v1"  # 改为老张API地址
)
```

### 第三步：调用模型
```python
# 一个接口，调用所有模型
models = [
    "gpt-4.1",               # OpenAI最新模型
    "claude-sonnet-4-20250514",  # Claude 4最新版
    "gemini-2.5-pro",        # Google Gemini
    "deepseek-v3"            # 国产模型
]

for model in models:
    response = client.chat.completions.create(
        model=model,
        messages=[{"role": "user", "content": "你好"}]
    )
    print(f"{model}: {response.choices[0].message.content}")
```
